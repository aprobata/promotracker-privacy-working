# PromoTracker Privacy & Data Deletion Policy

## Data Collection
PromoTracker collects only the minimum data required to provide app functionality.
This may include:
- Email address (for account login)
- App usage data (for improving features)

No sensitive personal data is sold or shared with third parties.

## Data Storage & Security
All data is transmitted securely over encrypted connections (HTTPS).
User data is stored securely using Supabase infrastructure.

## Account & Data Deletion
Users can request deletion of their account and all associated data at any time.
